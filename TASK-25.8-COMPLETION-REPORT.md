
#### 3.2 添加快捷回复使用说明
在"使用说明"章节添加了详细的快捷回复功能说明：

**核心功能**:
- 📝 多种模板类型
- 📁 分组管理
- 🔍 快速搜索
- 🌐 翻译集成
- 📊 使用统计
- 💾 导入导出
- 🎯 批量操作
- 🖱️ 拖拽排序

**使用方法**:
1. 点击右侧面板的"快捷回复"图标（💬）
2. 在操作面板中浏览和使用模板
3. 点击"编辑管理"打开管理界面创建和编辑模板
4. 选择发送模式（原文/翻译后）
5. 点击"发送"直接发送，或"输入框提示"插入到输入框

**账号隔离**:
- 每个 WhatsApp 账号拥有独立的快捷回复配置
- 切换账号时自动加载对应的模板库
- 数据完全隔离，互不影响
