# 激活码管理后台

独立的管理后台应用，用于生成和管理应用激活码。

## 功能特性

- ✅ **图形化界面**：友好的Web界面，无需命令行
- ✅ **批量生成**：一次可生成多个激活码
- ✅ **灵活配置**：支持自定义有效期和设备数
- ✅ **预设模板**：试用版、标准版、高级版、永久版
- ✅ **激活码管理**：查看、复制、删除激活码
- ✅ **统计信息**：实时显示激活码使用情况
- ✅ **完全独立**：与客户端完全分离

## 启动方式

### Windows

双击运行 `start.bat` 或在命令行中执行：

```bash
cd admin
start.bat
```

### macOS/Linux

```bash
cd admin
chmod +x start.sh
./start.sh
```

## 使用说明

### 1. 生成激活码

1. 选择激活码类型（试用版、标准版、高级版、永久版）
2. 设置有效天数
3. 设置最大设备数
4. 输入生成数量
5. 添加备注（可选）
6. 点击"生成激活码"

### 2. 管理激活码

- **查看**：所有生成的激活码都会显示在列表中
- **复制**：点击"复制"按钮将激活码复制到剪贴板
- **删除**：点击"删除"按钮移除激活码

### 3. 激活码类型

| 类型 | 有效期 | 适用场景 |
|------|--------|----------|
| 试用版 | 30天 | 用户体验 |
| 标准版 | 365天 | 普通用户 |
| 高级版 | 730天 | VIP用户 |
| 永久版 | 100年 | 终身授权 |

## 数据存储

激活码数据保存在 `admin/activation-codes.json` 文件中。

## 安全注意事项

1. **私钥保护**：
   - `privateKey.pem` 文件必须严格保密
   - 不要分享给任何人
   - 定期备份到安全位置

2. **激活码管理**：
   - 记录每个激活码的分配情况
   - 定期清理不需要的激活码
   - 妥善保管激活码

## 技术架构

- **框架**：Electron 39.1.1
- **前端**：HTML + CSS + JavaScript
- **加密**：RSA 2048位签名
- **存储**：JSON文件
